Политика конфиденциальности

Настоящая политика описывает порядок обработки персональных данных, которые пользователь передает через формы сайта.

1. Какие данные мы получаем

Через формы сайта могут передаваться имя, номер телефона, тип объекта и комментарий пользователя.

2. Для чего используются данные

Данные применяются только для обратной связи, уточнения задачи и подготовки предложения по оценке.

3. Где хранятся данные

Заявки отправляются через автоматизированный webhook-поток и могут сохраняться в CRM, Google Sheets и рабочих системах компании.

4. Передача третьим лицам

Персональные данные не передаются третьим лицам вне рамок законных требований и технологической обработки заявки.

5. Права пользователя

Пользователь вправе запросить уточнение, изменение или прекращение обработки данных, направив обращение по контактам сайта.

6. Контакты по вопросам данных

По вопросам обработки персональных данных можно связаться по телефону или по адресу info@ocenka-group.ru.
